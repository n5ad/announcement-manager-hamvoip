 After some, numerous, requests came in, I have decided to modify my Announcement Manager to work in Supermpon Hamvoip
it is compatible with hamvoip 6.1 through the latest 7.4+
 here are the steps
```
 cd /etc/asterisk/local
```
 ```
sudo wget https://raw.githubusercontent.com/n5ad/announcement-manager-hamvoip/refs/heads/main/announcement-manager-hamvoip.sh
```
```
sudo bash announcement-manager-hamvoip.sh
```
